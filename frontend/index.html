Zcash Shielded Transaction Viewer

Decrypt and view shielded transaction details using your viewing key. All decryption happens locally in your browser.

How Your Privacy Is Protected

100% Client-Side Decryption: Your viewing key is processed entirely in your browser using WebAssembly. It is never transmitted to any server.

Official Zcash Libraries: We use the official Zcash Rust cryptographic libraries (zcash_primitives, orchard, sapling-crypto) compiled to WebAssembly.

Minimal Backend: The only server communication is fetching the raw transaction data from a Zcash node. Your viewing key is never sent.

Open Source: This tool is open source. You can audit the code, build it yourself, or run it entirely offline.
